# System Tests

Системные (нефункциональные) сценарии в `system_tests/test_system_quality.py`.

## Категории
- `load`: тесты нагрузочного профиля
- `stability`: тесты стабильности при многократных циклах
- `recovery`: тесты восстановления после ошибок/отказов
- `volume`: тесты с большими объемами данных

## Набор сценариев
1. Массовое создание сессий одним пользователем (load)
2. Многократные start/stop циклы (stability)
3. Fallback API времени и восстановление нормальной работы (recovery)
4. Большой объем сессий и выборки по месяцам (volume)
5. Недоступность API расписания не ломает основной flow (recovery)
6. Очередь scheduler не растет без ограничений (stability)
7. Последовательности действий нескольких пользователей (load)
8. Большой набор достижений (volume)
9. Ошибка логина с последующим восстановлением (recovery)
10. Пакетная генерация токенов (load)

## Запуск
```bash
python -m pytest system_tests -m system -v
```
