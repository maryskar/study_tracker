# Integration Tests

Интеграционные сценарии взаимодействия модулей в `integration_tests/test_integration_workflows.py`.

## Количество сценариев
- 10 завершенных пользовательских кейсов
- есть позитивные и негативные сценарии

## Локальный запуск
```bash
pytest integration_tests -m integration -v
```

## Особенности
- используется изолированная тестовая SQLite БД
- внешние API изолируются через mock
- планировщик таймера заменен на тестовую заглушку
